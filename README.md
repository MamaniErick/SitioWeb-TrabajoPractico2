# 🌍 Mega Turismo - Sitio Web de Agencia de Viajes

Este proyecto es una página web desarrollada como trabajo práctico para la materia **Programación y Servicios Web**.

## Descripción

Mega Turismo es una página web informativa de una agencia de viajes ficticia que permite mostrar destinos turísticos, información de agencias y contacto.

El objetivo del proyecto es aplicar conocimientos de:
- HTML5
- CSS3
- Diseño responsivo
- Estructura de sitios web
- Uso de Git y GitHub

---

##Tecnologías utilizadas

- HTML5
- CSS3
- Git & GitHub
